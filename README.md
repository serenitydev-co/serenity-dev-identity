# serenity-dev-identity

Static brand identity presentation for Serenity Dev.

## Canonical assets
- `assets/serenity-dev-symbol.svg`: source SVG symbol used in the presentation
- `assets/png/`: final PNG exports used by the project
- `assets/png/symbol-square-color.png`: large square color variant
- `assets/png/symbol-square-black.png`: large square monochrome variant

Process notes and review documents for this project live outside this directory.
